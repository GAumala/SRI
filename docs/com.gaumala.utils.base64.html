encode
multimethod
Multimethod para codificar valores a Base64. Solo String y byte array son soportados. Devuelve un String en Base64.


(encode "hello world!")
; => "aGVsbG8gd29ybGQh"
(encode (.getBytes "hello world!"))
; => "aGVsbG8gd29ybGQh"
